# Codex 기능별 작업 시작용 프롬프트

이 폴더는 큰 앱 작업을 기능별로 나눠서 새 Codex 대화에서 이어가기 위한 프롬프트 모음입니다.

사용법:

1. 새 Codex 대화를 엽니다.
2. 작업하려는 기능 파일을 엽니다.
3. 파일 내용을 그대로 붙여넣습니다.
4. Codex가 해당 기능만 읽고 수정하도록 진행합니다.
