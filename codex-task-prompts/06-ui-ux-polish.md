이 프로젝트는 React Native + Expo로 만든 기타 악보 연습 앱이야.
현재 폴더는 `C:\Users\kis42\OneDrive\Desktop\real coding\digitalhighschoolproject` 이야.

이번 대화에서는 **UI/UX 정리만** 작업해줘.

중요 기준:
- iPad 기준 앱이다.
- 홈 화면은 스크롤 없이 보기 좋아야 함.
- 홈의 1, 2, 3, 4 메뉴는 화면을 꽉 채우는 느낌이어야 함.
- 연습 화면은 악보를 편하게 보는 것이 가장 중요함.
- 연습 화면 아래에는 시작하기/이어서/정지 같은 핵심 버튼만 두는 것이 좋음.
- BPM과 조심해서 연주할 구간은 악보 옆 보조 패널에 둠.
- 영어 UI는 가능하면 한국어로 바꿔야 함.

목표:
- 홈, 연습 화면, 취약 부분, 성취도, 튜닝 화면의 전체 시각 균형 정리.
- 버튼 위치와 크기 조정.
- iPad 키보드나 화면 크기에 가려지는 입력창 없게 하기.
- 기능 로직은 최소한으로만 건드리기.

작업 방법:
- 먼저 `App.tsx`의 렌더링 JSX와 StyleSheet를 읽어줘.
- 기능 로직, Firebase, MusicXML 파싱, 피치 판정은 최대한 건드리지 마.
- 수정 후 `npm.cmd run typecheck`를 실행해줘.
